# ARTERIA

***SETUP***
First, install all requirements with
```
pip install -r requirements.txt
```

***RUN***
First, change in *run.py* path to interpreter. Then enter this commands:
```
chmod +x run.py
./run.py
```

Then open 127.0.0.1:5000 in your browser.
